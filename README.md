# Gmail Inbox Sender
> Get latest 5 mails and send mail with file

## Installation

OS X & Linux:

```sh
sudo apt update && upgrade
pip3 install -r requirements.txt
apt install qttools5-dev-tools
apt install python3-pyqt5.qtsql
apt install python3-pyqt5
```
